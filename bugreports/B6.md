# Б6 — Поле «Добавить права» подсвечивается зелёным сразу после отправки документов

- **ID:** Б6  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».  
2. Кликнуть в поле «Права» / «Добавить права».  
3. Заполнить: Имя Иван, Фамилия Иванов, дата рождения 12.06.2001, номер 01 01 123456.  
4. Снять фокус с поля.  
5. Нажать активную кнопку «Добавить».  
6. В открывшемся окне нажать «Принять» или крестик.

## Ожидаемый результат

Рамка поля «Добавить права» зелёная, зелёная галочка справа внутри поля появляется только после успешной проверки документов.

## Фактический результат

Рамка подсвечивается зелёным сразу после отправки документов на проверку (Chrome, Firefox).

- Браузер: Google Chrome v.144.0.7559.133
