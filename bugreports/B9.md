# Б9 — В подписи к «Светомузыка» отсутствует слово «Доступно»

- **ID:** Б9  
- **Приоритет:** Стандартный  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».  
2. Найти пункт «Светомузыка».  
3. Просмотреть подпись под пунктом.

## Ожидаемый результат

Подпись: «Доступно в тарифе «Роскошный»».

## Фактический результат

Подпись: «В тариф «Роскошный»» (без «Доступно»).

- Браузер: Google Chrome v.144.0.7559.133
