# Б12 — При выбранном «Повседневном» на карте отображаются авто только этого тарифа

- **ID:** Б12  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».  
2. Навести курсор на иконки автомобилей по очереди и дождаться подсказки.

## Ожидаемый результат

На карте отображаются свободные автомобили всех тарифов.

## Фактический результат

Отображаются только автомобили тарифа «Повседневный».

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
