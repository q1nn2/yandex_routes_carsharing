# Б10 — На кнопке «Ввести права» вместо «Добавить права» при заполненной оплате

- **ID:** Б10  
- **Приоритет:** Стандартный  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть «Способ оплаты».  
2. Добавить карту: номер 123456789123, код 12.  
3. Снять фокус с поля.  
4. Нажать «Привязать».

## Ожидаемый результат

Кнопка содержит текст «Добавить права и забронировать» и описание маршрута.

## Фактический результат

Текст «Ввести права и забронировать» и описание маршрута.
