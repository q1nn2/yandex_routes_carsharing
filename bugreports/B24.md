# Б24 — Пробелы в «Номер карты» не проставляются автоматически

- **ID:** Б24  
- **Приоритет:** Стандартный  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. «Способ оплаты» → «Добавить карту».  
2. Ввести номер `000000000001`, снять фокус.

## Ожидаемый результат

Отображение с группировкой, например `0000 0000 0001`.

## Фактический результат

Номер остаётся без автоматических пробелов: `000000000001`.
