# Б33 — Исчезает форма бронирования при удалении адресов при заполненных правах и оплате

- **ID:** Б33  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Добавить права: Иван Иванов, 12.10.2000, 01 01 123456.  
2. Добавить карту (например номер 000000000001, код 12).  
3. Удалить значение поля «Откуда».  
4. Удалить значение поля «Куда».  
5. Нажать кнопку бронирования.

## Ожидаемый результат

Кнопка с текстом «Забронировать»; по нажатию — окно «Машина забронирована» (согласно сценарию в таблице).

## Фактический результат

Исчезает форма бронирования.

- ОС: Windows 11 Pro 21H2, 1920×1080
