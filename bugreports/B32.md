# Б32 — Без оплаты по клику «Забронировать» открывается «Машина забронирована»

- **ID:** Б32  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Добавить права: Иван Иванов, 12.10.2000, 01 01 123456.  
2. Нажать кнопку бронирования.

## Ожидаемый результат

Текст кнопки «Добавить оплату и забронировать»; по нажатию открывается окно добавления способа оплаты.

## Фактический результат

Открывается окно «Машина забронирована».
