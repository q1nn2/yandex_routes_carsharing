# Б2 — Отсутствует иконка идущего человека в блоке описания тарифа

- **ID:** Б2  
- **Приоритет:** Желательный  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. Ввести «Откуда»: Хамовнический вал, 18.  
3. Ввести «Куда»: Усачева, 3.  
4. Режим «Свой», транспорт «Каршеринг».  
5. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».

## Ожидаемый результат

Соответствие макету Figma (иконка идущего человека).

## Фактический результат

В Google Chrome и Firefox иконка не отображается.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
