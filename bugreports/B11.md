# Б11 — Иконки свободных автомобилей на карте ориентированы параллельно (не по направлению дороги)

- **ID:** Б11  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».

## Ожидаемый результат

Маленькие синие иконки расположены по карте согласно местности и адресам, ориентация по макету.

## Фактический результат

Иконки расположены параллельно друг другу «носом на север» (Chrome, Firefox).

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
