# Б4 — Отсутствует маркер перед описанием фич в блоке тарифа

- **ID:** Б4  
- **Приоритет:** Желательный  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».

## Ожидаемый результат

Текст «• видеорегистратор • зарядка для телефона» серого цвета под изображением авто.

## Фактический результат

Перед словом «видеорегистратор» отсутствует маркер (Chrome, Firefox).

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
