# Б21 — В поле «Номер карты» можно ввести пробелы

- **ID:** Б21  
- **Приоритет:** Стандартный  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. «Способ оплаты» → «Добавить карту».  
2. Ввести в «Номер карты»: `12 3456 78 9101`.

## Ожидаемый результат

Пробелы не сохраняются; в поле остаётся `123456789101`.

## Фактический результат

Пробелы вводятся и отображаются.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
