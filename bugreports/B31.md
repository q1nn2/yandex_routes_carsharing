# Б31 — Не открывается окно «Добавление прав» по клику на кнопку бронирования

- **ID:** Б31  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Добавить карту: номер 123456789123, код 12.  
2. Нажать кнопку бронирования.

## Ожидаемый результат

Открывается окно «Добавление прав».

## Фактический результат

Окно «Добавление прав» не открывается.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
