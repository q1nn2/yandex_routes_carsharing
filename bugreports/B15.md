# Б15 — По клику на свободный автомобиль не появляется чёрная плашка с маркой

- **ID:** Б15  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».  
2. На карте навести курсор на иконки машин и кликнуть.

## Ожидаемый результат

Иконка увеличивается, над ней чёрная плашка с маркой авто.

## Фактический результат

Иконка не реагирует на клик ожидаемым образом / плашка не появляется.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
