# Б14 — Клик по иконке автомобиля на карте не обновляет данные на левой панели

- **ID:** Б14  
- **Приоритет:** Критический  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Кликнуть по тарифу «Повседневный».  
2. На карте навести курсор на иконку машины и кликнуть.

## Ожидаемый результат

При клике по иконке информация об автомобиле на левой панели обновляется.

## Фактический результат

Информация на левой панели не меняется в соответствии с выбранной машиной.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
