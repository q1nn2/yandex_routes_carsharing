# Б30 — Текст кнопки: «Ввести права» вместо «Добавить права»

- **ID:** Б30  
- **Приоритет:** Желательный  
- **Статус:** Открыт

## Предусловия

1. Перейти на тестовый стенд.  
2. «Откуда»: Хамовнический вал, 18; «Куда»: Усачева, 3.  
3. Режим «Свой», «Каршеринг».  
4. Нажать «Забронировать».

## Шаги воспроизведения

1. Добавить карту: номер 123456789123, код 12.  
2. Нажать кнопку бронирования.

## Ожидаемый результат

Текст кнопки: «Добавить права и забронировать» + строка маршрута.

## Фактический результат

Текст: «Ввести права и забронировать» + строка маршрута.
